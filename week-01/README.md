## Catatan Pembelajaran

Pada pertemuan pertama, saya mempelajari pengenalan dasar mengenai Reverse Engineering. Saya memahami bahwa reverse engineering adalah proses menganalisis sebuah software atau binary untuk memahami cara kerja internalnya tanpa melihat source code asli.

Saya juga mulai memahami bahwa reverse engineering banyak digunakan dalam berbagai bidang seperti software analysis, vulnerability research, malware analysis, dan cybersecurity. Dari materi awal ini saya mengetahui bahwa proses analisis biasanya dibagi menjadi dua pendekatan utama, yaitu static analysis dan dynamic analysis.

Selain itu, saya dikenalkan dengan beberapa tools yang umum digunakan dalam reverse engineering seperti Ghidra, IDA Free, dan Wireshark. Saat ini saya mulai memahami fungsi dasar dari masing-masing tools, meskipun saya masih perlu belajar lebih jauh mengenai cara penggunaannya secara langsung dalam proses analisis binary.
