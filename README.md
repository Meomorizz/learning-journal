Learning Journal Reverse Engineering

Repository ini berisi catatan pembelajaran saya selama mengikuti perkuliahan Reverse Engineering dan Malware Analysis.

Tujuan dari repository ini adalah untuk mendokumentasikan proses belajar secara bertahap selama satu semester, mulai dari konsep dasar hingga praktik analisis binary dan malware.

Seluruh catatan ditulis berdasarkan hasil pembelajaran pribadi, diskusi di kelas, praktik laboratorium, serta eksplorasi mandiri menggunakan berbagai tools reverse engineering.

Topik yang dipelajari meliputi:

Fundamental Reverse Engineering
Static Analysis
Dynamic Analysis
Binary Inspection
Malware Analysis
Crackme Challenges

Setiap jurnal mingguan berisi hal yang saya pelajari, pemahaman yang saya dapatkan, serta kendala atau hal yang masih ingin saya eksplorasi lebih lanjut.
